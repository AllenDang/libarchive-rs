pub fn set_perm(&mut self, perm: u32) -> Result<()>

Set the file permissions

On platforms where permissions are stored as u16 (macOS, Windows, BSD), this will return an error if perm > 0xFFFF. Standard Unix permissions (0o777) are always safe on all platforms.

§Platform Notes

The underlying mode_t type varies by platform:

  • Linux (64-bit): u32
  • macOS/BSD/Windows: u16
  • Android: varies by architecture (u32 on 64-bit, u16 on 32-bit)
